? | What is MathLib?

MathLib - is a library that helps in solving mathematical calculations.

MathLib Functionality

The main task of Matlab is to help solve mathematical calculations, and for this there are several methods that are divided into groups.

Arithmetic methods

- add(a, b): Returns the sum of two numbers.

 add(a, b) {
   return a + b;
 }

- subtract(a, b): Returns the difference between two numbers.

 subtract(a, b) {
   return a - b;
 }

- multiply(a, b): Returns the product of two numbers.

 multiply(a, b) {
   return a * b;
 }

- divide(a, b): Returns the result of dividing a by b. If b is 0, it returns "Not divisible by 0".

 divide(a, b) {
  if (b === 0) {
   return "Not divisible by 0";
  } else {
   return a / b;
  }
 }

Trigonometric methods

-- sin(angleInRadians): Returns the sine of the angle in radians.

static sin( angleInRadians) {
  return Math.sin(angleInRadians);
}

-- cos(angleInRadians): Returns the cosine of the angle in radians.

 static cos( angleInRadians) {
   return Math.cos(angleInRadians);
 }

-- tan(angleInRadians): Returns the tangent of the angle in radians.

 static tan(angleInRadians) {
   return Math.tan(angleInRadians);
 }

Additional methods

--- squareRoot(x): Returns the square root of the number x.

 squareRoot(x) {
   return Math.sqrt(x);
 }

--- power(base, exponent): Returns the result of raising the base number to exponent.

 power(base, exponent) {
   return Math.pow(base, exponent);
 }

Additional information

* Note: All methods use mathematical functions from the JavaScript Standard Library (Math).
